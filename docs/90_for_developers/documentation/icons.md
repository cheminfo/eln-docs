# Insert icons

The documentation project includes the `react-icons` package, which allows you to add numerous icons directly in the text. All the icons are available [here](https://react-icons.github.io/react-icons/).

To add icons in the documentation, add the following text at the top of the file:

```
import { FaCogs } from 'react-icons/fa';
import { AiFillBug } from 'react-icons/ai';
```

Here `FaCogs` should be replaced with the list of all the icons you want to insert.

- `<FaCogs />` <FaCogs />
- `<AiFillBug />` <AiFillBug />

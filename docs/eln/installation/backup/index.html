Backing up the system

Since CouchDB is an append-only database, using a rsync/rsnapshot-based solution is a safe option. For large database you may be interested the rsync option --append that will only append data onto shorter files and therefore not copy the full database.

Alternatively, you can use the CouchDB replication feature to another server. You can conveniently set up replication from the Fauxton graphical interface at the _utils endpoint.
